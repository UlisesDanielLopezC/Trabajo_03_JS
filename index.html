Enlaces sin leer

Antes de acceder a un enlace, de ser posible, asegurarse de que se trata de una web oficial y no de un HTML copiado para hacerte ingresar datos personales, se pueden identificar estos casos leyendo el enlace e investigandolo o comparandolo en un buscador.

Informacion de mas

Diferentes sitios recolectan diversa informacion, entre ellas se pueden encontrar las conexiones, pero para mantener cierta seguridad en la navegacion web, es preferible proveer los menos datos posibles en sitios de caracter informal donde sea posible.
